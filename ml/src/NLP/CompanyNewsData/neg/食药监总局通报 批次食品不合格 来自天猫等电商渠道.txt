






　　多种网售年货被爆不合格
　　食药监总局通报8批次食品不合格，7批来自电商渠道
　　来源：南方都市报

　　网购年货需当心。就在离春节还剩1周时间之际，多个线上渠道在售的传统年货被爆不合格。
　　据国家食品药品监督管理总局6日披露，其近期组织抽检粮食加工品、肉制品、饼干、乳制品、饮料、特殊膳食食品、罐头、茶叶及相关制品和糖果制品等9类食品711批次样品，抽样检验项目合格样品703批次，不合格样品8批次。
　　值得注意的是，8批次不合格食品，7批来自电商渠道。其中，还包括广氏菠萝味碳酸饮料、嘉士利果乐果香果酱味夹心饼干等年货类商品。
　　线上渠道不合格明显高于线下
　　总体而言，此次抽查结果显示国内春节前夕年货类商品的质量整体还算不错。
　　除了特殊膳食食品21批次中有不合格样品3批次外，粮食加工品、肉制品、乳制品、饼干、饮料分别抽查了117批次、75批次、101批次、98批次和71批次，均只有一批次不合格样品。此外，罐头74批次，茶叶及相关制品36批次，糖果制品118批次样本，均未检出不合格。
　　而细看样本来源，可以看到线上渠道不合格明显高于线下渠道。
　　以过年期间热销的水磨年糕为例，此次被查出的不合格样品就来自电商平台。
　　不过，阿里巴巴相关媒体负责人昨日向南都记者表示，天猫和淘宝对不合格产品一直都有严格的规定。对于不合格商品都是做立即下架处理。“阿里巴巴非常重视食品安全，并积极参与食品安全的预警和交流，早在2011年就已成立专门团队管控食品安全。不仅在入驻环节严核商家经营资质，也在商品流通环节严核厂名、厂址、生产许可证等资质要素。”该位负责人进一步说。
　　“广氏菠萝啤”
　　涉事产品已召回
　　在CFD A的通报中，南都记者留意到，“天猫超市”销售的广氏菠萝味碳酸饮料出现菌落超标的情况，据悉，被抽检出现超标产品标称为“广州啤酒厂委托佛山市三水区隐雪食品有限公司（下文简称：隐雪食品）生产”。
　　在CFD A抽检报告中，抽样产品菌落总数5次检出值分别为6200CFU /m l、8100CFU /m l、5000CFU /ml、710 0 0 0 C F U /m l、5600CFU /m l，不符合“菌落总数5次检测结果均不得超过10000CFU /m l且至少3次检测结果不超过1 0 0CFU /m l”的国家标准规定。
　　据公开资料显示，被抽检的广氏菠萝味碳酸饮料即“广氏菠萝啤”，是广州轻工旗下广州啤酒厂的龙头产品，其中“广氏”品牌于2012年获“中国优质特色啤酒产品”称号、2013年荣获广州十大手信产品及2014年获得广东老字号称号等荣誉。
　　对于被C F D A抽检出菌落超标情况，南都记者向广州啤酒厂致电采访，对方相关负责人表示，收到CFD A通报后，涉及菌落超标的产品（仅1批次）已召回封存，另外该负责人表示，公司方面已开展相关自查工作，并未发现其他批次产品存在菌落超标等问题。
　　多名国内啤酒负责管控产品生产的负责人向南都记者表示，按照目前啤酒的生产工艺而言，出现菌落总数超出国家标准规定的情况较为“罕见”，那么，如果产品中被检出菌落超标又因何而起？对此，前述品控负责人均表示，如果产品出现菌落超标情况，有可能与原料生产和储存设备、灌装时容器清洗或管道容器等方面卫生不严有关。
　　对此，上述广州啤酒厂内部人士向南都记者表示，从目前初步自查情况来看，暂未发现委托生产方隐雪食品在生产流程中存在问题，初步判断涉事产品菌落超标“有可能是流通过程中仓储或运输过程”所致。
　　涉事饼干企业回应
　　问题发生在非生产环节
　　除了饮料，饼干类产品也有广东品牌卷入其中。
　　国家食品药品监督管理总局组织共抽检饼干样品98批次，覆盖15个生产省份（及海外）的61家（生产）企业。其中抽检饼干98批次，不合格样品数1批次，检出不合格的检测项目为霉菌。不合格产品是天猫伽盛食品专营店（经营者为深圳市伽盛实业有限公司）在天猫（网站）销售的标称广东嘉士利食品集团有限公司生产的嘉士利果乐果香果酱味夹心饼干，霉菌检出值为190C F U /g，比国家规定标准（不超过50C FU/g）高出2 ．8倍。检验机构为四川省食品药品检验检测院。
　　昨日南都记者在淘宝及天猫均未找到伽盛食品专营店，而在深圳市伽盛实业有限公司阿里巴巴网站平台，也没有看到嘉士利及果乐果香相关产品。嘉士利相关负责人回应南都记者采访时称，此次涉及的产品霉菌超标是发生在非生产环节，“江门药监局和省药监局曾出文件向国家局申请复检，说明非生产环节问题”。据悉 ，果 乐 果 香 是 嘉 士 利2012年底推出，属明星产品，曾一度占夹心饼干系列大部分销售额。
　　国家食药监局曾发布霉菌超标相关的资料称，霉菌超标可能是加工用原料受霉菌污染，或者是由于产品存储、运输条件控制不当引起霉菌滋生。霉菌污染可使食品腐败变质，破坏食品的色、香、味，降低食品的食用价值。
